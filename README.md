##### 例外，关于友链的添加方式，友链的代码是内嵌进了单页模块，当建立了友链单页时，需要在编辑内填写如下字段完成添加
```
<li>网站名称</li>
<li>网站网址</li>
<li>网站图标</li>
<li>网站简介</li>
```
##### 如上的添加方式重复即可添加更多友链
